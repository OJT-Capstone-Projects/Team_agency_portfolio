# Team Agency Portfolio

A professional agency portfolio website built with pure HTML, CSS, and JavaScript.

## Tech Stack
- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript

## Branches
| Branch | Description |
|--------|-------------|
| `main` | Stable base branch |
| `feature/header-hero` | Header, Navigation, Hero & About sections |

## Getting Started
Clone the repo and open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/OJT-Capstone-Projects/Team_agency_portfolio.git
cd Team_agency_portfolio
```
