# Team Agency Portfolio

A professional agency portfolio website built with pure HTML, CSS, and JavaScript — no frameworks, no libraries.

## Tech Stack
- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (CPR syllabus)

## Project Structure

```
Team_agency_portfolio/
│
├── index.html               # Main HTML file
│
├── css/                     # All stylesheets
│   └── header-hero.css      # Header, Nav, Hero & About styles
│
├── js/                      # All JavaScript files
│   └── header-hero.js       # Header, Nav, Hero & About logic
│
└── assets/                  # Static assets
    ├── images/              # Project & team images
    └── icons/               # Custom icons & SVGs
```



## Getting Started

```bash
git clone https://github.com/OJT-Capstone-Projects/Team_agency_portfolio.git
cd Team_agency_portfolio
```

Open `index.html` in your browser — no build step required.
